以前我一想到習慣養成，腦中浮現的詞常常是「自律」「堅持」「意志力」。可是這幾年慢慢感覺到，如果一件事情每次都需要靠意志力硬撐，通常很難真正持續。畢竟人不是機器，生活也不會永遠替我們保留一段乾淨、安靜、精力充沛的時間，讓我們優雅地完成所有好習慣。

更奇妙的是，有些事情明明很好，卻怎麼樣都做不下去。比如學英文、瑜伽、冥想、運動、閱讀、寫作，這些聽起來都很正確，也常常被包裝成「你應該做」的事。可是知道重要，不代表它真的會進入生活。相反地，有些事情沒有人逼我，我卻已經自然做了很久，像是晨間隨筆、散步、口述記錄、寫生活日誌、和 AI 對話整理想法，甚至這段時間持續經營自由人生實驗室、拍攝 YouTube 影片。

所以我開始想：習慣做不下去，真的只是因為我懶、拖延、不夠自律嗎？還是有可能，這件事根本沒有被設計成容易發生？甚至更深一層，它其實還沒有真正成為我的選擇？

<!--more-->

這篇文章想整理我目前對習慣養成的理解。它不是要教人變成超級自律的人，而是想提出一個比較貼近我自己的公式：

> **可持續習慣 = 容易發生 + 真的像我**

或者更完整地說：

> 一個習慣要長久，不只要在行為設計上容易發生，也要在意義認同上真的符合自己的價值、身分與生活方向。

如果只有行為設計，沒有意義認同，習慣容易變成外部任務。
如果只有意義認同，沒有行為設計，習慣又容易停在空想。

這兩件事要一起成立，習慣才有機會慢慢長成生活的一部分。

## 先反過來看：哪些事我已經自然做了很久？

談習慣時，我們常常只盯著「我做不到什麼」。可是另一個更有意思的方向是：反過來拆解，那些我已經做得到的事情，為什麼能持續？

以我自己來說，這幾件事情已經比較自然地嵌進生活裡：

| 已經自然持續的事   | 為什麼比較容易持續                 |
| ---------- | ------------------------- |
| 晨間隨筆       | 通常接在早起、喝咖啡、開電腦之後，已經形成生活節奏 |
| 散步         | 身體需要走動，也和思考、放鬆、觀察環境連在一起   |
| 散步口述       | GETSEED 錄音卡降低摩擦，可以邊走邊記錄念頭 |
| 生活日誌       | 我本來就有記錄者傾向，想保存生活痕跡        |
| 和 AI 對話    | 能幫我把模糊念頭整理出結構，也符合我的學習模式   |
| 拍自由人生實驗室影片 | 雖然還在實驗，但它接上了記錄、創作、學習與公開表達 |

這些事情不一定每天都完美執行，但它們已經不像外來任務，而比較像我生活的一部分。它們有兩個共同點：

第一，它們的開始門檻不高。
第二，它們和「我是什麼樣的人」有關。

這就是我後來整理出的兩層：**行為設計**與**意義認同**。

## 第一層：行為設計，讓習慣容易發生

BJ Fogg 的行為模型對我很有啟發。他提出一個簡潔的公式：

> **B = MAP**

也就是一個行為 Behavior 要發生，需要三個條件同時到位：

| 元素            | 意思         | 可以問自己的問題        |
| ------------- | ---------- | --------------- |
| Motivation 動機 | 我想不想做？     | 我為什麼想做這件事？      |
| Ability 能力    | 這件事容不容易做到？ | 這件事現在對我來說會不會太難？ |
| Prompt 提示     | 什麼情境提醒我開始？ | 我會在什麼時候自然想起來做？  |

這個模型讓我想到一件事：
習慣養成不是把人變成機器，而是替自己鋪一條比較容易走的小路。

很多時候，我們把所有希望押在動機上。看完一本書、聽完一場演講、刷到一支熱血影片，就覺得「我也要開始改變人生！」可是幾天後熱情退去，事情又停下來。

動機很重要，但動機不可靠。
如果一件事只能靠打雞血啟動，它大概撐不了太久。

能力也很重要。這裡的能力不只是「我會不會」，更包含「這件事對我來說夠不夠容易」。比如 BJ Fogg 曾經用牙線的例子說明，他知道牙線對牙齒健康重要，所以動機不是問題；真正卡住的是，整口牙都用牙線清潔太麻煩。於是他把行為縮小成「每天只清一顆牙」，又找更細、更好用的牙線，讓行為變得容易。

提示則是讓行為被觸發的情境。手機 App、遊戲、小紅點、通知，其實都很會利用提示。它們不需要你每次重新決定，只要時間到了、畫面跳出來，你就被提醒去做某件事。

既然外部產品可以這樣設計我們的行為，我們當然也可以把這套邏輯拿回來，用在自己真正想要養成的習慣上。

## 工具一：行為集群清單，把願望拆成行為

很多人一開始寫下的是願望，不是行為。

例如：

| 願望            | 不是行為的原因        |
| ------------- | -------------- |
| 我想變健康         | 太抽象，不知道下一步做什麼  |
| 我想持續創作        | 是方向，不是具體行動     |
| 我想變有錢         | 是結果，不是每天可執行的動作 |
| 我想更自由         | 是價值核心，但需要落地行為  |
| 我想成為 YouTuber | 是身分或目標，不是最小行動  |

真正能改變現狀的是行為。

例如「我想變健康」可以拆成：

* 去健身房鍛鍊
* 不喝含糖飲料
* 晚上十點前睡覺
* 早上六點前起床
* 睡前三十分鐘不用手機
* 餐後立刻刷牙
* 使用牙線
* 做飲食記錄

「我想持續創作」也可以拆成：

* 晨間隨筆
* 散步口述
* 拍一鏡到底影片
* 和 AI 對話
* 整理逐字稿
* 寫網站文章
* 做 YouTube 說明欄
* 回顧日誌素材

這一步的重點不是馬上判斷哪個最好，而是先把模糊願望翻譯成一群具體行為。

先不要管它們可不可行，也不要急著篩選。這一步只是把抽象願望拆開，讓它們變成可以被觀察、被設計的行為。

## 工具二：黃金行為焦點地圖，不要從最漂亮的習慣開始

列出行為之後，下一步不是選最漂亮、最理想、最符合成功人士形象的習慣，而是選最有可能活下來的行為。

這就是 Fogg 提到的「焦點地圖」對我最有啟發的地方。

它用兩個維度來看行為：

| 維度  | 問題              |
| --- | --------------- |
| 影響力 | 這個行為對我的願望有多大幫助？ |
| 可行性 | 現在的我真的做得到嗎？     |

放進四象限後，右上角就是最適合開始的黃金行為：

> **高影響 × 高可行 = 黃金行為**

例如「變健康」這個願望：

| 行為     | 影響力 | 可行性 | 判斷           |
| ------ | --- | --- | ------------ |
| 去健身房鍛鍊 | 高   | 低   | 很好，但目前不容易持續  |
| 不喝含糖飲料 | 高   | 中   | 重要，但需要替代方案   |
| 早睡早起   | 高   | 高   | 已經相對可行，可持續強化 |
| 少用手機   | 中高  | 低   | 有幫助，但目前卡住    |
| 餐後立刻刷牙 | 高   | 高   | 很適合先開始       |
| 飲食記錄   | 中   | 中   | 可做，但未必是最核心   |

這裡最重要的觀念是：

> 不要從最漂亮的習慣開始，要從最可能活下來的習慣開始。

這句話對我自己的創作也很有用。

如果我一開始就要求自己每天拍一支精緻影片、完整剪輯、做好字幕、設計封面、優化標題，那我大概很快就會陣亡。可是如果我先允許自己一鏡到底、不剪輯、用簡報或口述讓內容先成立，那它就有機會活下來。

我現在已經拍到三十幾集自由人生實驗室影片，靠的不是突然變得超級自律，而是把拍影片這件事做成比較低摩擦、比較能持續的版本。

3. 它需要很多準備嗎？

## 工具三：微習慣配方，錨點、微行為、慶祝

找到黃金行為之後，不要立刻把它變成龐大任務，而是把它設計成微習慣配方。

公式是：

> 當我＿＿＿＿之後，
> 我將＿＿＿＿，
> 然後我＿＿＿＿。

也就是三個部分：

| 元素  | 意思         | 例子              |
| --- | ---------- | --------------- |
| 錨點  | 已經穩定存在的舊行為 | 刷牙後、泡咖啡後、吃完晚餐後  |
| 微行為 | 小到容易開始的新行為 | 寫一句話、清一顆牙、讀一頁   |
| 慶祝  | 立刻給自己成功感   | 打勾、微笑、說一句「我做到了」 |

例如健康習慣可以設計成：

> 當我吃完晚餐後，
> 我立刻刷牙，
> 然後對自己說：「今天有照顧牙齒。」

寫作習慣可以設計成：

> 當我早上泡完咖啡後，
> 我打開日誌寫一句話，
> 然後對自己說：「今天也有留下痕跡。」

影片創作也可以是：

> 當我準備好簡報後，
> 我按下 OBS Studio 錄影，
> 然後對自己說：「先完成，再優化。」

我以前可能會覺得「慶祝」有點尷尬，做了一件小事也要慶祝嗎？可是後來想想，慶祝不一定要很浮誇。它只是讓大腦知道：這件事值得被重複。

人如果只會檢討自己做不好，卻不允許自己承認「我開始了」「我做到了」，那要長期持續其實很難。

### 小練習：寫下你的微習慣配方

請填入：

> 當我＿＿＿＿＿＿＿＿之後，
> 我將＿＿＿＿＿＿＿＿，

記得：
微行為要小到你幾乎沒有理由拒絕。
慶祝要簡單、立即，而且真的能讓你有一點成功感。

## 工具四：容易做分解，把大石頭拆成小石頭

如果微習慣還是做不下去，不要急著責怪自己。
可以繼續做「容易做分解」。

我把它整理成三個方向：

| 方法   | 問題            | 例子           |
| ---- | ------------- | ------------ |
| 縮小   | 這件事還能不能更小？    | 寫一千字 → 寫一句話  |
| 移除障礙 | 哪個步驟最卡？       | 不剪輯，先一鏡到底    |
| 提前準備 | 能不能先幫未來的自己鋪路？ | 運動鞋先放好、文件先開好 |

縮小行為很好理解。
想寫一千字太難，就先寫一句話。
想運動三十分鐘太難，就先做一個伏地挺身。
想讀一小時太難，就先讀一頁。

移除障礙對我特別有用。以拍 YouTube 來說，我最討厭的就是剪輯和上字幕。一想到後面還有這些事情，我可能連拍都不想拍。所以我現在先把剪輯刪掉，採用一鏡到底，讓影片先能完成。

這不是偷懶，而是 MVP，最小可行版本。

先做出能運作的版本，讓它活下來，再決定要不要優化。這比一開始就追求完美，結果永遠無法開始，要實際多了。

提前準備則是降低啟動摩擦。想運動，就先把運動鞋放好；想寫作，就先把文件開好；想錄音，就把 GETSEED 放在固定位置；想拍片，就先準備簡報或小卡。

我越來越相信，很多習慣不是靠意志力，而是靠摩擦力管理。

## 反轉：為什麼小事已經很小，還是做不下去？

講到這裡，好像只要降低難度、找到提示、縮小行為，就能解決所有習慣問題。

可是事情沒有那麼簡單。

有些時候，即使行為已經小到不能再小，我們還是不想做。

例如：

* 每天只背一個英文單字，還是不想背
* 每天只做一個伏地挺身，還是不想做
* 每天只寫一句話，還是不想寫
* 只走五分鐘，還是不想出門

如果事情已經這麼小，我還是不想做，那問題可能就不是難度，而是我根本沒有真正認同這件事。

這就是第二層：**意義認同**。

行為設計處理的是：

> 我做不做得到？

意義認同處理的是：

> 我到底要不要讓這件事進入我的人生？

## 第二層：意義認同，從「我應該」到「這就是我」

很多習慣停在「我應該」。

我應該學英文，因為出國旅行會用到。
我應該運動，因為健康很重要。
我應該早起，因為成功人士都早起。
我應該寫作，因為輸出很重要。
我應該經營 YouTube，因為現在是自媒體時代。

可是「應該」不一定會長成真正的行動。它常常只是外部標準。

《原子習慣》裡我覺得最有力的部分，是身分認同。當一個行為只是外部任務，它很難長久；但當它和「我是什麼樣的人」連在一起，它就不只是待辦事項。

例如：

| 外部任務語言  | 身分認同語言            |
| ------- | ----------------- |
| 我應該寫作   | 我是記錄者，所以我會記錄生活    |
| 我應該學習   | 我是學習者，所以我會研究新東西   |
| 我應該拍影片  | 我是創作者，所以我會把經驗轉成作品 |
| 我應該測試工具 | 我是體驗者，所以我願意實驗新方法  |
| 我應該做流程  | 我是管理者，所以我會整理時間與系統 |

李笑來曾經談過戒菸的例子。他不是一直說「我正在戒菸」，而是把語言切換成「我是個不抽菸的人」。這兩句話差很多。

「我正在戒菸」聽起來像我很想抽，但我正在忍耐。
「我不抽菸」則是在說：這件事和我無關，因為我不是那種人。

這不是單純話術，而是身分切換。

一個人一旦相信「我不是那種人」或「我就是這種人」，行為就不需要每一次都重新談判。

## 凱根：這是我的選擇，還是外部標準？

談到意義認同，我覺得羅伯特・凱根的成人心智發展理論也很有幫助。

這裡不完整介紹他的理論，只抓和習慣最有關的三個階段。

| 階段     | 習慣背後的動機          | 可能的狀態           |
| ------ | ---------------- | --------------- |
| 社會化心智  | 大家說這樣好，所以我做      | 容易被按讚、評價、外部標準牽動 |
| 自我主導心智 | 這符合我的價值，所以我做     | 內心有羅盤，不完全依賴外部回饋 |
| 自我轉化心智 | 我也能反思自己的原則是否還服務我 | 不被固定身分或習慣綁住     |

社會化心智並不是壞事。人本來就需要被看見，也需要回饋。可是如果一個習慣完全依賴外部評價，一旦沒有人按讚、沒有人留言、沒有人稱讚，就很容易失去力量。

自我主導心智比較像是：我做這件事，是因為它符合我自己的價值。

我寫作，是因為我重視記錄。
我拍影片，是因為我想練習公開表達。
我經營自由人生實驗室，是因為它是我的生活實驗容器。

可是即使到了自我主導，也還有一個風險：身分認同可能反過來綁住自己。

如果我說「我是高效能的人」，是不是就不能休息？
如果我說「我是日更者」，是不是一天沒更新就覺得自己失敗？
如果我說「我是創作者」，是不是什麼生活都要變成素材？

這時候就需要自我轉化心智。

我能做，也能不做。
我選擇做，但我不被它綁架。
我知道自己為什麼做，也知道什麼時候該調整。

所以我很喜歡這句話：

> 成熟不是永遠堅持，而是知道自己為什麼做，也知道什麼時候該調整。

## 用自由人生實驗室檢查：拍影片這件事值得嗎？

這次錄製影片，其實發生了一些有點哭笑不得的事。

昨天，我不小心按到麥克風靜音，後半段影片完全沒有聲音。今天早上，我拍了將近一小時，結果因為資料夾路徑名稱改過，檔案根本沒有存進去。換句話說，我等於又重拍了一次。

如果單純從效率來看，這真是很荒謬。

可是我還是重新錄了。

這時候就很適合用意義認同的三道問題檢查：

### 一、價值問題：我真的相信它值得嗎？

拍影片對我來說，至少有幾種價值：

* 練習口說表達
* 訓練整理概念的能力
* 留下此刻生命的紀錄
* 有可能幫助某個正在路上的人
* 即使沒有人看，我自己也在過程中學到東西

當我把影片錄下來，它就以某種方式留在世界上。也許三年、五年後，我可以回頭看見現在的自己正在想什麼、怎麼說話、怎麼跌跌撞撞地做實驗。

這件事是有價值的。

### 二、身分問題：它符合我想成為的人嗎？

拍影片其實符合我很重視的幾個角色：

| 角色  | 拍影片如何符合                         |
| --- | ------------------------------- |
| 管理者 | 需要管理時間、素材、流程、器材與表達              |
| 學習者 | 為了拍片，要研究理論、工具與方法                |
| 創作者 | 把想法轉成作品                         |
| 記錄者 | 留下此刻的自己                         |
| 體驗者 | 親自測試 YouTube、OBS Studio、低摩擦創作流程 |

所以它不只是「我應該經營 YouTube」，而是它真的接上了我想成為的人。

### 三、主權問題：這是我要做，還是別人要我做？

外部世界當然會影響我。現在大家都在談自媒體、AI 時代、公開輸出、日更、個人品牌，我不可能完全不受影響。

可是自由人生實驗室的影片，對我來說並不只是跟風。它更像是我在試著把多年來累積的筆記、口述、閱讀、散步思考，用另一種形式公開輸出。

它是我要做，不只是別人要我做。

這個差別很重要。

## 沒有人看見，還願意做嗎？

有一個很殘酷、但很有效的問題是：

> 如果沒有人看見、沒有人稱讚，我還願意繼續做嗎？

我不能假裝自己完全不在乎流量，也不能假裝完全不需要被看見。人本來就需要回饋，需要知道自己不是對著空氣說話。

可是現實是，我現在的影片觀看數很少，稱讚也不多。儘管如此，我還是已經拍到三十幾集。

這至少說明，它不完全是靠外部評價撐著。

我不需要假裝自己不想被看見。
但我也可以承認，即使現在還沒有很多人看見，我仍然願意繼續做。
因為它對我自己已經有價值。

這可能就是意義認同真正開始生效的地方。

## 可持續習慣＝容易發生 + 真的像我

最後，我想把這篇收束回一開始的公式：

> **可持續習慣 = 容易發生 + 真的像我**

只有行為設計，沒有意義認同，習慣容易變成外部任務。
只有意義認同，沒有行為設計，習慣容易停在空想。

真正可持續的習慣，需要同時滿足兩件事：

| 條件   | 問題                       |
| ---- | ------------------------ |
| 容易發生 | 它夠小嗎？摩擦夠低嗎？有提示嗎？接得進生活嗎？  |
| 真的像我 | 它符合我的價值嗎？符合我的身分嗎？是我的選擇嗎？ |

理想可以遠大，但落地必須微小。
意義可以深，但行為要簡單。
真正重要的事，不能只放在心裡，也不能被做成壓迫自己的任務。

它需要被低摩擦地安放進生活。

## 最後的檢查：一年後，它會讓我更自由，還是更焦慮？

最後，我想用一個問題收尾：

> 如果我持續做這件事情一年，它會讓我更自由，還是更焦慮？

以 YouTube 為例，如果我一直追流量、追熱點、追別人有沒有看，那它很可能會讓我更焦慮。過去很多創作嘗試，可能就是這樣無疾而終。

但如果它讓我練習表達，學會工具，留下紀錄，累積作品，讓自由人生實驗室慢慢變成生活的一部分，那它就有可能讓我更自由。

不是所有好習慣都適合放進人生。
那些真正值得留下的習慣，應該讓我更靠近自己想要的生活，而不是變成新的束縛。

習慣養成不是把自己訓練成一台機器。

它是先找到真正重要的事，再用簡單、低摩擦的方式，把它安放進生活裡。

而如果那件事同時容易發生，又真的像我，也許它就會慢慢長成一個可持續的習慣。
