三分鐘熱度也許是另一種探索的可能性

Wednesday, Apr 15, 2026 | 8 minute read | Updated at Wednesday, Apr 15, 2026

@ garrolan

最近這兩天是定期評量日。不過老實說，我心裡倒還好，並沒有真的把學生考試這件事擺在精神世界的頭等大事位置。

那麼，我的注意力到底放在哪裡了呢？

我得承認，自己其實是個挺三分鐘熱度的人。

一件事情，剛開始的時候可能熱情滿滿、興致盎然，覺得很新鮮、很有趣，想要一頭栽進去研究。可是，一旦做到某個程度，開始感覺自己似乎摸清楚它了、掌握得差不多了，那種原本強烈的興奮感就會慢慢退去，興趣也跟著轉移。

這樣一講，簡直有點像渣男（笑）。

總像是對某些新鮮的、難攻不破的對象感到強烈興趣，想靠近、想理解、想挑戰；可一旦真的熟悉了，又慢慢失去熱情。每一任都以為自己是真命天女，但最後似乎都留不住他。

當然，這不是什麼渣男理論。真要說的話，我覺得更接近心流理論，或者探索與利用理論，在學習、生活與工作上的某種體現。

所謂有興趣，很多時候其實就是進入了某種心流體驗。而之所以會對一件事情感到有趣，往往是因為它剛好落在我的學習區、挑戰區裡。它不是那種太簡單、太無聊、只剩機械重複的舒適區；也不是那種完全搞不懂、燒腦到令人退縮的恐慌區。

它是那種：雖然有點難，但還摸得到邊；雖然還不懂，但又覺得自己好像有可能搞懂。那種區域，最讓人上癮。

當遊戲快做完時，興奮感也差不多退了

像昨天，我用 Codex 把地圖探索 RPG 遊戲收了尾，也部署上架到 Netlify 上面。

整體來說，心情比較接近一種：「啊……差不多終於要結束了。」
而不是一開始那種：「咦？這到底怎麼弄的？我完全搞不懂啊，但好像很值得研究看看。」

也就是說，對我而言，製作這款遊戲的興奮點其實已經下降了。我的熱情並不是消失，而是已經悄悄往別的方向轉移。

那麼，下一個方向是哪裡？

最近這兩天，我開始嘗試 Google 的桌面型 agent──Antigravity。這名字也真夠長的，所以我後來都直接簡稱它 ANTI。

我會去碰它，倒也不是因為有什麼非學不可的使命感，而比較像是一種自然延伸的探索：既然我最近因為 Codex 開始接觸 agent，那麼是不是也可以順便摸摸看別家的 agent 到底是怎樣的東西？反正 Google AI Pro 我也已經訂了，不用白不用。

至於 Claude，雖然看起來 cowork 功能不錯，但老實說，我真的已經訂太多 AI 工具了。微軟 365、GPT、Google、GET 筆記（GETSEED錄音卡）、得到 App聽書電子書……光這些就夠多了。既然如此，Claude 暫時就先不考慮。我的注意力，便自然落到 OpenAI 和 Google 這兩家的 agent 工具上。

我想知道的是：它們到底能做什麼？差異在哪？實際用起來，感覺又是怎樣？

我學新工具的方法，似乎一直都很像

一開始接觸 Antigravity，其實只是想先搞懂它的介面與功能。

上週五，我在學校筆電安裝了它。那時候也就只是安裝而已，沒有真的深入碰，只覺得這東西看起來挺有距離感。後來再進一步登入 Google 帳號，一進去之後，整個介面幾乎都是英文。對我這種英文不算好的人來說，認知負荷明顯就上升了。

於是我又回到自己熟悉的方式：先截圖，再問GPT。

我把畫面一頁一頁截下來，問他每一個彈窗到底在說什麼、整個介面有哪些功能、那些看不懂的名詞是什麼意思，甚至還問能不能切換成中文。GPT也的確很詳細地幫我解釋，還教我去搜尋繁體中文介面的翻譯工具，讓我安裝來用。

這整個過程，後來想想，其實跟我前陣子摸索 Codex 很像。

那時候我也是把 Codex 的介面與功能一頁一頁截圖，發給GPT，讓他幫我解釋：這個按鈕是什麼？那個功能有什麼差別？哪個名詞到底代表什麼意思？如果還有不懂的，就繼續追問。可能一整個上午，甚至一整天，就花在把工具的用途慢慢摸清楚這件事上。

我發現，自己學新工具時，似乎很常採用這種方式：先不要假裝自己懂，而是老老實實承認看不懂，然後透過截圖、提問、拆解，一點一點把介面與概念翻譯成我能理解的東西。

比起抽象學習，我其實更適合任務驅動

而再往前一步，我通常還會把學習拉進「任務驅動」的模式裡。

以 Antigravity 為例，正式進入之後，它要我連接一個資料夾作為 workspace。系統不知為何，自動跳出一個我有點陌生、又有點眼熟的資料夾。我愣了一下才想起來：那是很久以前，大概 2025 年 9 月，我曾經讓 Codex 幫我做過的一個個人網站專案，後來因為我自己完全搞不懂，也看不懂它到底在做什麼，最後就這樣無疾而終，靜靜躺在舊筆電的 D 槽裡。

久到我幾乎忘了它的存在。

結果反而是 Antigravity 先把它挖出來了。

既然如此，我乾脆就把這個資料夾撿回來。反正它是當下我唯一還認得的專案資料夾，那就讓 Antigravity 去讀讀看，看看它到底能辨識出什麼、能不能跟我說明：這到底是什麼東西？有什麼用途？我現在還能不能重新把它用起來？

原本只是想學工具，結果慢慢變成：既然都要學 Antigravity，不如就找一個專案來邊做邊學。這樣至少不只是研究介面，還能順手做出一點東西來。畢竟對我來說，輸出往往能倒逼輸入。

所以整件事就演變成：藉著重新搭建個人網站這個任務，來學習如何操作 Antigravity。

而且這個方向，我其實也會忍不住想：如果 Codex 額度充足，它是不是也能一起幫忙？只是最近前面做遊戲時，額度消耗得差不多了，直到 4/17 之前只剩下二十幾趴。那麼，暫時把注意力轉移到 Antigravity 上，倒也挺合理。

當我想做網站時，真正的問題其實不是技術

當然，既然要重新做個人網站，很快就會碰到另一層問題：

那我到底要在這網站上放什麼？

而且更麻煩的是，不只是放什麼，還包括：AI 到底要怎樣介入這件事，才能幫助我，而不是取代我、淹沒我的聲音？

我甚至也會進一步想到：那麼，這跟我之前用 Google Blogger 寫《自由人生實驗室》，到底差在哪裡？如果名稱一樣、內容也類似，那這個 GitHub 網站存在的意義是什麼？只是平台換了而已嗎？

又或者，我是不是其實可以把 Blogger 上那邊的文章重新整理、重新包裝，慢慢搬到這個網站上？

可是，一想到這裡，新的問題又來了。

我要怎樣做，才不會變成想太多、做太少？
又或者反過來，變成只是機械勞動，一直重複做事，卻沒有真正思考？

我不喜歡兩種極端

我最近其實很明顯地感覺到，自己不喜歡兩種極端。

第一種，是過度發散、過度想像，甚至有點偏離主題。

像前幾天散步時，我一度開始天馬行空地暢想：欸，其實我可以把用 Codex 製作教學遊戲的過程錄成 YouTube，發布出去耶。那我是不是應該買個耳麥？這樣錄影時收音比較方便？然後下一秒，又開始想：我是不是太自以為是了？是不是有點狂妄？再接著，甚至開始構思：那我要打造怎樣的人設？建立怎樣的 IP 形象？

到這裡我就警覺了。

因為這些想法聽起來好像很積極，但仔細看，其實根本已經偏離原本的重點太遠。事情都還沒開始，腦中就先長出一大堆包裝、定位、人設與設備需求。這很容易讓人卡在計畫裡，卻沒有真的開始。

第二種，則是過度機械、過度重複，只剩執行，沒有探索。

那種感覺就像只是為了做而做，大量、批次、重複性地處理某些事情，沒有思考，沒有試驗，沒有新鮮感。久了會讓我覺得，人好像被異化成工具。我很不喜歡那種狀態。

說到底，我好像真正喜歡的，是中間那一段：既不是空想，也不是機械勞動，而是能夠一邊做、一邊想、一邊試、一邊調整方向。

我愈來愈喜歡 MVP 這個概念

最近我其實愈來愈喜歡一個詞：MVP，最小可行性產品。

就是有棗沒棗，先打三竿。不要一開始就想做到多完美，而是先弄出一個最小可行版本，跑跑看，看看能不能動。行得通就繼續，不行就修改。

回頭看，我前幾款教學遊戲，很多時候其實也都差不多是這樣做出來的。並不是一開始就有一套多完整的藍圖，而是先做出一個能跑的東西，然後邊玩邊改，邊試邊修。反而在這樣的過程裡，慢慢摸索出一點方向，也累積出一些心得。

所以我也會想，也許這次《自由人生實驗室》的網站，也可以用同樣的思路來做。

把它當成 2.0 版本。

雖然名稱一樣，但底層使用的資源已經不同了。這次不是 Blogger，而是 GitHub 網站；這次也不是我自己單打獨鬥亂摸，而是開始嘗試讓 Antigravity、GPT，甚至其他 AI 工具一起參與，幫助我慢慢搭出一個新的版本。

它不需要一開始就很完整，也不必急著證明自己多厲害。也許先做出一個可用、可看的版本，邊用邊修，才比較像我真正做事的方式。

我可能正在尋找的，不只是網站，而是一種方法

寫到這裡，我愈來愈覺得，自己真正想找的，好像不是某一個工具、某一個平台、某一套技術而已。

我在找的，其實更像是一種適合自己的方法。

一種適合我的聲音。
一種適合我的創作型態。
一種適合我的輸出節奏。

我希望它夠輕鬆，不至於帶來太大的負擔；
又希望它能夠持續，不是三天打魚、兩天曬網；
我也希望它不是完全不經過我的大腦，只是讓 AI 全自動代工；
可同時又希望，最後長出來的東西，能夠真的代表我。

這樣講起來，的確有點既要又要還要。
一整個特別任性（笑）。

但也許，正因為我現在還沒找到，所以才更需要慢慢試。

我隱隱感覺，這件事情不適合躁進。它不太像那種設好目標、定好時程、照表操課就能穩穩完成的任務。它比較像是一條需要在思考與研究、試錯與調校之中，慢慢走出來的路線。

說到底，我現在也許還在探索：
怎樣的方式，才能既保有我的聲音，又讓 AI 真正成為輔助；
怎樣的節奏，才不會讓我變成只會想很多卻不動手，或者只剩下埋頭苦幹卻失去靈魂。

而《自由人生實驗室》這個網站，也許正是一個新的實驗場。

尾聲

已經快六點了，現在時間是 05:48。今天的晨間隨筆，先聊到這裡。

不過寫到最後，我倒是忽然意識到一件有趣的事：我似乎隱隱預設，這篇隨筆很可能之後會被我拿去整理成網站文章。於是，在對話語氣上，也就少了一點那種完全對自己說話的感覺，反而多了一些像是在向某個對象交代的意味。

這件事，我覺得自己也需要有意識。

因為那很可能又牽涉到另一個問題：當我開始意識到文字可能會被公開，我的聲音是不是也會慢慢發生偏移？

這件事，或許之後還可以再慢慢觀察。

總之，現在的我，仍然在尋找一種真正適合自己的方式。
適合我的聲音，適合我的型態，也適合我這種會在探索中不斷轉移熱情的人。

也許，我其實不是單純三分鐘熱度。
我只是總在尋找，下一個還值得我投入心神去探索的東西。
