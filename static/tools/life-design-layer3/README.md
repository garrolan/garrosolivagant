# 日常意義工具箱｜How to Live a Meaningful Life

這是一組靜態 HTML 多頁式網頁，用來練習從日常提取意義。

## 頁面

- `index.html`：首頁與入口選擇
- `wonder.html`：照片上傳、感受標籤、詩意文字小卡，支援下載 PNG
- `coherence.html`：價值花園、價值定義、重視度／實踐度一致性檢查
- `flow.html`：小事心流練習、注意力入口、可選三分鐘倒數、Flow 紀錄
- `community.html`：連結花園，記錄陪伴型／合作型／形成型連結
- `meaning-map.html`：整合 Wonder、Coherence、Flow、Community 的摘要與意義句

## 使用方式

直接用瀏覽器打開 `index.html` 即可。資料會暫存在使用者瀏覽器的 `localStorage`，沒有後端資料庫。

## 設計方向

自然清新、手帳感、低壓力互動；避免做成心理測驗、KPI 儀表板或效率工具。

## 概念來源

概念來自 Bill Burnett 與 Dave Evans 的 Life Design 系列，尤其是《How to Live a Meaningful Life》的 Wonder、Coherence、Flow、Community 四個意義入口。本專案為學習與展示用途，互動題目與網頁流程已重新設計。
