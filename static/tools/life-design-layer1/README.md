# GARRO自由人生實驗室｜第一層人生設計工具箱

這是一組純前端靜態網頁，可直接本機開啟或放到 GitHub Pages。

## 內容

- `index.html`：首頁與工具入口
- `dashboard.html`：生活儀表盤
- `compass-work.html`：人生羅盤｜工作觀
- `compass-life.html`：人生羅盤｜人生觀
- `journal.html`：好時光日誌
- `aeiou.html`：AEIOU 觀察法
- `problem-finding.html`：卡住點盤點
- `summary.html`：第一層總結

## 技術

- HTML / CSS / JavaScript
- 不使用後端資料庫
- 不使用 AI API
- 使用瀏覽器 `localStorage` 儲存資料

## 注意

資料只存在使用者自己的瀏覽器中。如果換裝置、清除瀏覽器資料，內容會消失。可在總結頁匯出 JSON 保存。
